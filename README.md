# Python-Platformer

## Setup
- At the top of VS Code, select:
  - "Show and run commands >"
  - "Python: Create environment..."
  - Venv
- At the top of VS Code, select:
  - "Show and run commands >"
  - "Terminal: Create new terminal"
- Run in terminal:
```
pip install -r requirements.txt
```

## Development
### New requirements

```
pip freeze > requirements.txt
```
